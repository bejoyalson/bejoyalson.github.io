About Me

I'm a full-stack developer with 7+ years of experience building production-grade web applications. My expertise spans backend systems (ASP.NET Core, REST APIs, microservices) and modern front-end development (React, TypeScript).

I'm passionate about:

  • Clean Architecture: Vertical slice organization, SOLID principles, separation of concerns
  • Performance: Query optimization, caching strategies, load testing, and profiling
  • Reliability: Automated testing, CI/CD pipelines, graceful error handling
  • Scalability: Database indexing, async/await patterns, distributed tracing

When I'm not coding, I contribute to open-source projects and write about software engineering best practices.
